# vue-cesium-platform
Vue结合Cesium的web端GIS平台



## 特性

-  Vue构建界面
- 集成GIS基础功能





## 快速开始

 下载源码，在根目录执行

- 安装依赖

```shell
npm install
```

- 运行

```shell
npm run dev
```



访问

http://127.0.0.1:3001
