# vue-cesium-platform
Vue结合Cesium的web端gis平台



## 初步效果



> 笔记本性能限制，运行Cesium温度飙到70度以上。所以平时开发时先开发界面，之后加载Cesium地球

![平台](images/平台首页.png)



![平台首页2](images/平台首页2.png)



## 当前实现的功能

- 组件化，模块功能分工
- 集成Cesium
- 操作面板可切换
- 操作面板增加关闭按钮

## 源码及如何运行

 下载源码，在根目录执行

- 安装依赖

```shell
npm install
```

- 运行

```shell
npm run dev
```
